# Ruby on Rails Tutorial: sample application

A twitter-like application for
[*Ruby on Rails Tutorial:
Learn Web Development with Rails*](http://www.railstutorial.org/)
by [Michael Hartl](http://www.michaelhartl.com/). You can create a user, post microposts, and follow or be followed by other users. 

[See it live!](http://aqueous-citadel-47985.herokuapp.com/)
